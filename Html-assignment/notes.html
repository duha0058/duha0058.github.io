HTML Tags

<div>
A block level container used to group elements for styling or layout purposes.
<a>
Defines a hyperlink. The href attribute specifies the URL of the page the link goes to.
<p>
Defines a paragraph of text. It automatically adds space before and after the paragraph
<em>
Used to emphasize text. By default, it renders the text in italics.
<blockquote>
Used to define a section that is quoted from\ another source. It typically renders the text in an indented format.
